# 🚀 AI Job Recommendation System

## 📌 Overview
This project is an AI-based job recommendation system that analyzes a user's resume and suggests suitable job roles. It also provides ATS scoring and skill improvement suggestions.

---

## 🔥 Features
- 📄 Resume Parsing (PDF)
- 🎯 Job Matching based on skills
- 📊 ATS Score Calculation
- 🧠 Skill Gap Analysis
- 📈 Skill Improvement Suggestions
- 🎨 Interactive UI using Streamlit

---

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- NLP (basic text processing)

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
